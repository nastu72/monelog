# monelog

簡単な収支が記録できる家計簿サービスです。（※レスポンシブ未対応）<br>
データベースにはMAMPのMySQLを使用しました。<br>

【使用言語】 PHP（生） / HTML / CSS<br>
【機能】 ユーザー登録・退会 / ログイン・ログアウト / パスワード変更・リマインド / 収支データ一覧・登録・編集・削除<br>
【制作時間】 37時間（17日）<br>
